Delete Space in String

ES 5

//去除空格 type 1-所有空格 2-前后空格 3-前空格 4-后空格 //ecDo.trim(' 1235asd',1) //result：1235asd //这个方法有原生的方案代替，但是考虑到有时候开发PC站需要兼容IE8，所以就还是继续保留 trim:function(str,type){ switch(type){ case 1: return str.replace(/\s+/g,""); case 2: return str.replace(/(^\s*)|(\s*$)/g,""); case 3: return str.replace(/(^\s*)/g,""); case 4: return str.replace() } }
