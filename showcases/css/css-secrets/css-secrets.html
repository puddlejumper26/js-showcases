box-shadow 多重投影
<div class="box-shadow"></div> .box-shadow{ width: 100px; height: 60px; margin: 25px; background: yellowgreen; box-shadow: 0 0 0 10px #655, 0 0 0 15px deeppink, 0 2px 5px 15px rgba(0,0,0,.6); }
  • box-shadow 它还接受第四个参数（称作“扩张半径”）， 通过指定正值或负值， 可以让投影面积加大或者减小
  • 它支持逗号分隔语法， 我们可以创建任意数量的投影
  • 投影的行为跟边框不完全一致， 因为它不会影响布局， 而且也不会受到 box-sizing 属性的影响
  • 上述方法所创建出的假“边框”出现在元素的外圈。 它们并不会响应鼠标事件， 比如悬停或点击。 如果这一点非常重要， 你可以给box-shadow 属性加上 inset 关键字， 来使投影绘制在元素的内圈。
  • 只能模拟实线边框（假设我们需要产生虚线边框效果， box-shadow 就没辙了）
  • 如果我们需要获得更多层的边框， 这种方案就是我们唯一的选择了

    outline 描边 | outline-offset
    <div class="outline"></div> .outline{ width: 200px; height: 100px; margin: 25px; background:rgb(31, 12, 49); outline: dashed white; outline-offset:-15px; }
  • outline-offset 属性来控制它跟元素边缘之间的间距， 这个属性甚至可以接受负值
  • 只适用于双层“边框”的场景， 因为 outline 并不能接受用逗号分隔的多个值

    background-position | background-origin
    Code Pirate
    <div class="background-position"></div> .background-position{ background: url(http://csssecrets.io/ images/code-pirate.svg) no-repeat bottom right #58a; background-position: right 20px bottom 10px; /* Styling */ max-width: 10em; min-height: 5em; padding: 10px; color: white; font: 100%/1 sans-serif; }
  • 背景图片跟右边缘保持 20px 的偏移量， 同时跟底边保持 10px 的偏移量
  • 在网页开发生涯中， 你很可能多次写过类似 background-position:top left;这个 top left 到底是哪个左上角？
  • 每个元素身上都存在三个矩形框border box（边框的外沿框）、 padding box（内边距的外沿框） 和 content box（内容区的外沿框）
  • 默认情况下， background-position 是以 padding box 为准的， 这样边框才不会遮住背景图片
  • background-origin(默认值是padding-box)的值改成 content-box（参见下面的代码）， 我们在 background-position 属性中使用的边角关键字将会以内容区的边缘作为基准
    • .background-position{ padding: 10px; background: url("code-pirate.svg") no-repeat #58a bottom right; /* 或 100% 100% */ background-origin: content-box; }

    calc() 方案
    Code Pirate
    <div class="calc"></div> .calc{ background: url(http://csssecrets.io/ images/code-pirate.svg) no-repeat bottom right #58a; calc(100% - 20px) calc(100% - 10px); /* Styling */ max-width: 10em; min-height: 5em; padding: 10px; color: white; font: 100%/1 sans-serif; }
  • 希望它有一个 100% - 20px 的水平偏移量， 以及 100% - 10px 的垂直偏移量
  • calc() 函数内部的 - 和 + 运算符的两侧各加一个空白符，否则会产生解析错误！这个规则如此怪异，是为了向前兼容：未来，在 calc() 内部可能会允许使用关键字，而这些关键字可能会包含连字符（即减号）。

    横条纹 | background:linear-gradient() | background-size
  • 一条基本的垂直线性渐变， 颜色从 #fb3 过渡到 #58a
    • <div class="hentw-1"></div> .hentw-1{ width: 200px; height: 100px; background: linear-gradient(#fb3, #58a); }

  • 试着把这两个色标拉近一点
    • .hentw-2{ width: 200px; height: 100px; background: linear-gradient (#fb3 20%, #58a 80%); }

  • 已经没有任何渐变效果了， 只有两块实色，各占据了 background-image 一半的面积。 本质上， 我们已经创建了两条巨大的水平条纹
    • .hentw-3{ width: 200px; height: 100px; background: linear-gradient (#fb3 50%, #58a 50%); }

    nach oben

  • 因为#fb3 70% 是先被执行的，所以#58a 50%就被忽略了
    • .hentw-4{ width: 200px; height: 100px; background: linear-gradient (#fb3 70%, #58a 50%); }

  • 通过 background-size 来调整其尺寸
  • 把这两条条纹的高度都缩小到了 15px。 由于背景在默认情况下是重复平铺的， 整个容器其实已经被填满了水平条纹

    波点 | background-image: radial-gradient
  • 径向渐变允许我们创建圆形、 椭圆， 或是它们的一部分
    • <div class="bodian-1"></div> .bodian-1{ width: 300px; height: 300px; background: #655; background-image: radial-gradient(tan 30%, transparent 0); background-size: 30px 30px; }
  • 生成两层圆点阵列图案， 并把它们的背景定位错开， 这样就可以得到真正的波点图案
    • <div class="bodian-2"></div> .bodian-2{ width: 300px; height: 300px; background: #655; background-image: radial-gradient(tan 30%, transparent 0), radial-gradient(tan 30%, transparent 0); background-size: 30px 30px; background-position: 0 0, 15px 15px; }

    <div class="tuxiangbk-1">CONTENTS </div> .tuxiangbk-1{ padding: 1em; border: 1em solid transparent; background: linear-gradient(white, white), url(http://csssecrets.io/images/stone-art.jpg); background-size: cover; background-clip: padding-box, border-box; }
  • 主要的思路就是在石雕背景图片之上， 再叠加一层纯白的实色背景
  • 为了让下层的图片背景透过边框区域显示出来， 我们需要给两层背景指定不同的 background-clip 值
  • 我们只能在多重背景的最底层设置背景色， 因此需要用一道从白色过渡到白色的 CSS 渐变来模拟出纯白实色背景的效果
  • 边框的图片有一种怪异的拼接效果,原因是 background-origin 的默认值是 padding-box,把linear-gradient(white, white)设置成linear-gradient(transprant, white)就能看出区别，包括下面的例子也这样进行设置
  • 实际上就是背景图片以平铺的方式蔓延到 border box 区域的效果
    • Bacon ipsum dolor amet eu adipisicing elit tongue ground round ex fatback proident kielbasa ham hock. Sausage beef before ribs aliquip t-bone mollit. Quis beef tri-tip sunt, cupim ut magna salami t-bone. Ut laboris bresaola ribeye biltong

    信封样式
    My border is reminiscent of vintage envelopes
    <div class="xinfeng">CONTENTS </div> .xinfeng{ padding: 1em; border: 1em solid transparent; background: linear-gradient(white, white) padding-box, repeating-linear-gradient(-45deg, red 0, red 12.5%, transparent 0, transparent 25%, #58a 0, #58a 37.5%, transparent 0, transparent 50%) 0 / 5em 5em; } 这个效果也可以通过 border-image 来实现 padding: 1em; border: 16px solid transparent; border-image: 16 repeating-linear-gradient(-45deg, red 0, red 1em, transparent 0, transparent 2em, #58a 0, #58a 3em, transparent 0, transparent 4em); 不过 border-image 方法存在一些问题。
  • 每当我们改变 border-image-slice 时， 都需要同时修改 borderwidth 来让它们相互匹配。
  • 由于我们不能在 border-image-slice 属性中使用 em 单位， 只能把边框厚度指定为像素单位。
  • 条纹的宽度需要在色标的位置信息中写好， 因此我们在改变条纹宽度时， 需要修改四处。

    蚂蚁行军边框
    marching ants
    <div class="ant">CONTENTS </div> @keyframes ants { to { background-position: 100% 100% } } .ant{ padding: 1em; border: 1px solid transparent; background: linear-gradient(white, white) padding-box, repeating-linear-gradient(-45deg, black 0, black 25%, transparent 0, transparent 50%) 0 / .6em .6em; animation: ants 12s linear infinite; font: 100%/1.6 Baskerville, Palatino, serif; }
  • “老式信封”技巧的一个变种
  • 把条纹转变为黑白两色， 并把边框的宽度减少至 1px
  • background-size 改为某个合适的值
  • background-position 以动画的方式改变为 100%
  • border-image只能通过GIF动画来完成同样的事情
